## 扇贝作业Checklist :
1. 打开后看到文章列表 （✅）

2. 点击列表某项，打开文章 （✅）

3. 文章界面有一个按钮，点击则在文章中高亮在单词列表中出现的单词 （✅）

4. 文章界面有一个 slide-bar ，从0到6（对应单词表中每个单词信息的 level )，可以拖动过滤高亮的单词。 （✅）

5. 比较流畅舒服的 UI 交互、酷炫的动画 （✅）

6. 使用CoreText框架 （✅）

7. 数据库存储 （✅）
